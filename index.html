Mac OS X

Rename photos in mac OS X:

Rename files scripts for mac OS X, it will take a bunch of files and will change their name according with the scheme:
aaaa.mm.dd.filename.sequence.jpeg
The date field is created using the file creation time, the filename is a text string requested by the script via dialog window, sequence a is a sequential number generated. The files are here:
  • workflow, with a dialog box will appear asking for the files to be renamed;
  • application, files can be dragged and dropped over the application;
  • Service(file to be placed in $HOME/Library/Services/) this will pick current selected files for rename
