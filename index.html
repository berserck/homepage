Personal projects

curstat: Currency Statistics

github & demo

This is a small application that will query the api api.fixer.io for the Forex rates and do some data manipulation with it.

I wanted the checkout the symfony framework, to see how the basic setup was done and how long to get a basic website up. In a couple of days I managed to deploy this small toy application that will collect the exchange rate of several currencies for the last 5 working days and it will calculate in this time what was the maximum, the minimum and the average value of the exchange rate.

Nothing too complicated, I just wanted to check a few features

  • Query REST api: I'm getting the data from api.fixer.io, the link Show data will print the raw json and obtained from fixer.io.
  • Make a responsive design page: the home page will present the results in a table form that will adapt to your screen size
  • Make a REST API endpoint: I also implemented an REST api where you can all get the results in api/stats, or you can query the currency you want using api/stats/[currency]. The only HTML methods implemented are GET.

The source code can be found in github.

MacOS X: Image manipulation

Rename photos in MacOs X

Rename files scripts for mac OS X, it will take a bunch of files and will change their name according with the scheme: aaaa.mm.dd.filename.sequence.jpeg

The date field is created using the file creation time, the filename is a text string requested by the script via dialog window, sequence a is a sequential number generated.

The files are here:

  • workflow, with a dialog box will appear asking for the files to be renamed;
  • application, files can be dragged and dropped over the application;
  • Service (file to be placed in $HOME/Library/Services/) this will pick current selected files for rename.

Rotate photos in mac OS X
